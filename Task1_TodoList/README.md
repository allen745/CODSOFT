# ✅ Task 1 — To-Do List (TaskFlow)

A production-grade To-Do List web app built with **Python & Streamlit**.

## 🚀 Features
- Add tasks with title, description, priority, category & due date
- Mark tasks as complete / undo
- Edit tasks inline
- Delete tasks
- Filter by status, priority, category
- Search tasks
- Sort by date, priority, due date
- Live stats dashboard (total, active, completed, overdue)
- Progress bar
- Overdue detection
- Persistent JSON storage
- Fully responsive UI

## 🛠️ Tech Stack
- Python 3.10+
- Streamlit
- JSON (file-based storage)

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🌐 Deploy on Render.com
1. Push this folder to GitHub
2. Go to render.com → New → Web Service
3. Connect your GitHub repo
4. Set build command: `pip install -r requirements.txt`
5. Set start command: `streamlit run app.py --server.port $PORT --server.address 0.0.0.0`
6. Deploy!
