# ✅ Task 1 — To-Do List (TaskFlow)

A production-grade To-Do List web app built with **Python & Streamlit**.

## 🚀 Features
- Add tasks with title, description, priority, category & due date
- Mark tasks as complete / undo
- Edit tasks inline
- Delete tasks
- Filter by status, priority, category
- Search tasks
- Sort by date, priority, due date
- Live stats dashboard (total, active, completed, overdue)
- Progress bar
- Overdue detection
- Persistent JSON storage
- Fully responsive UI

## 🛠️ Tech Stack
- Python 3.10+
- Streamlit
- JSON (file-based storage)

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
