# 🧮 Task 2 — Calculator (CalcPro)

A production-grade Calculator web app built with **Python & Streamlit**.

## 🚀 Features
- Basic arithmetic: +, -, ×, ÷, power
- Scientific functions: sin, cos, tan, log, ln, sqrt, x², x³, 1/x, eˣ, |x|, floor, ceil
- Constants: π, e
- Memory functions: MC, MR, M+, M-
- Last answer (ANS) recall
- Calculation history (last 10)
- Clear history
- Sign toggle (+/-)
- Percentage calculation
- Backspace
- Two modes: Basic & Scientific

## 🛠️ Tech Stack
- Python 3.11+
- Streamlit

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🌐 Deploy on Render.com
1. Push this folder to GitHub CODSOFT repo
2. Go to render.com → New → Web Service
3. Root Directory: Task2_Calculator
4. Build: `pip install -r requirements.txt`
5. Start: `streamlit run app.py --server.port $PORT --server.address 0.0.0.0`
